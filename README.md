* [License](#license)

## Usage

Add `ccs811-rs` to `Cargo.toml`

```toml
[dependencies]
ccs811-rs = "^0.1.0"
```

or

```toml
[dependencies.ccs811-rs]
git = "https://github.com/iohe/ccs811-rs"
```
